Accessibility

Like localization, accessibility is an important aspect of Firefox copy creation. Similarly, writing well for the web – short sentences, clear hierarchy, non-ambiguous language – supports accessibility efforts. Additional considerations include the use of descriptive and meaningful alt text. You can learn more about designing for users who rely on assistive technologies here.

It is also important to be fair and respectful when we talk about all Firefox users, regardless of how they use our products to access the Web. Read about removing bias in accessibility language here.
