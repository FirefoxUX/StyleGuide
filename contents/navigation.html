Primary Navigation

Primary navigation is accessed via panels that are anchored to the browser frame.

Panels are transient UI surfaces that are anchored to an action item. They can contain notifications, menus, lists and other more complex UI widgets. Use panels for notifications and UI based on primary actions. For example a menu anchored to a toolbar item. These uses differ from context menus which are typically flat lists of secondary actions that happen when right-clicking an item.
