Other Button-like things

Some things are like buttons, but not quite.

Buttons

Button

This is just a button. Click it.

<button>Button</button>
Default Button

This is the default button. Click it or hit enter.

<button class="default">Button</button>
Footer Button

This is a footer button. Kick it.

<button class="footer">Button</button>
Default Footer Button

This is the default footer button. Kick it or hit enter.

<button class="default footer">Button</button>
