Welcome to the Firefox Style Guide

This guide is targetted at three main groups:

  1. Add-on Authors who want to write Firefox-y add-ons.
  2. Designers who want to make mockups of new features that look like they belong in Firefox
  3. and Firefox Engineers who want to make sure that the new features they’re adding fit in with the rest of the application.

Each of the sections on the left contains the pieces you might use, as well as example code you can include in your own project, and the rationale behind the designs, so feel free to click around and discover what’s there.

We’ve also collected all the recommendations for Add-on Authors into a single page, for ease of reference.
